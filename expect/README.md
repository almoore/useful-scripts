# Expect Scripts

Automated login scripts using [expect](https://core.tcl-lang.org/expect/index) for non-interactive SSH and MySQL sessions.

## Scripts

| Script | Description |
|--------|-------------|
| `ssh_expect` | Automated SSH login with configurable host, port, user, and password |
| `mysql_expect` | Automated MySQL login with configurable host, port, user, and password |

## Prerequisites

- `expect` (usually pre-installed on macOS, `apt install expect` on Debian/Ubuntu)
